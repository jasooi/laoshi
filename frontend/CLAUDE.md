# CLAUDE.md

This file provides guidance to Claude Code when working with frontend code in this folder.

## Design guide
- **Design Philosophy**: Clean, minimal interface with warm, muted color palette
- **Design System**:
  - **Primary Color**: Sage (#6B8F71) — buttons, active states, links
  - **Accent Colors**: Coral (#D4715E) for errors/warnings, Amber (#C4973B) for mid-range states
  - **Tints**: Sage Tint (#EDF2EE), Coral Tint (#FDF0ED), Amber Tint (#FBF5E8)
  - **Background**: Warm Offwhite (#FAFAF8)
  - **Text**: Warm Black (#2A2A28) primary, Warm Muted (#8A8A86) secondary
  - **Borders**: Warm Gray (#E8E5E0)
  - **Fonts**: Inter (UI), Lora (CJK characters)
  - **Shadows**: Minimal, soft shadows for depth
  - **Border Radius**: 8-12px for cards, 6px for buttons
  - **Spacing**: Consistent 4px/8px/16px/24px scale
  - **Icons**: Minimal, line-style icons
